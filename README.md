# Team 3 Final Project
## Project Name
Any Recipe Goes

## Target Audience
Those looking for delicious recipes that they can make at home

## Features and Functions
- Recipe search
- Favorite recipes
- Store own recipes (local storage)

## Project Requirements
### Detailed Form/Local Storage
Input/retrieve user recipes  
Other "form" for search function
### Data/API/json
JSON file that holds all "default" recipes/recipe information
### Drop-Down Menu
Mainly use for navigation - Splash/Home, List of all recipes/search function, Favorites page. 
### CSS Animations
Hover on buttons/elements changes box shadow or color, etc.
